# Handlingsplan

## Fase 1: Pitch-klar (0-4 uger)

Disse kræver INGEN kodeændringer — forberedelse til møder:

1. **Pitch-deck** med nøgletal fra noegletal.md
2. **Demo-video** af appen med professionelt dashboard
3. **DPA-skabelon** udarbejdet med jurist
4. **GDPR-roadmap** (gdpr-compliance.md) som viser compliance-plan
5. **Prissætning**: Definer pris pr. kommune (se prismodel.md)

## Fase 2: MVP for kommunal pilot (4-12 uger)

1. **MitID-integration** (største tekniske opgave)
2. **Audit-logging** implementering
3. **Brudnotifikationsprocedure** dokumenteret
4. **Supervisor-rolle** for ledere i familieretshuset
5. **Department-level adgang** (ikke kun individuel sagsbehandler)

## Fase 3: Fuld kommunal version (3-6 måneder)

1. Felt-niveau kryptering
2. Dataportabilitet (eksport)
3. Data-retention automatisering
4. Integration med kommunale systemer (SSO via Azure AD)
5. Digital signering af dokumenter

## Fase 4: Skalering (6-12 måneder)

1. Integration med Borger.dk
2. Integration med domstolssystemer
3. DPIA (Data Protection Impact Assessment)
4. Årlig sikkerhedsaudit
5. Penetrationstest
