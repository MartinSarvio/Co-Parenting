# Huska — Pitch-strategi til danske kommuner & familieretshuse

Huska er det primære digitale værktøj for co-parenting-familier. Målet er at skabe indtjening via kommunale aftaler, hvor sagsbehandlere og familieretshuse bruger appen som supplement til samarbejdskurser og sagsbehandling.

## Hvorfor kommunerne bør bruge Huska

**1. Reducerer sagsbehandlingstid**
- Struktureret kommunikation → færre misforståelser → hurtigere løsninger
- Beslutningslog med godkendelsesworkflow → dokumenteret enighed
- Bytteanmodninger håndteres digitalt i stedet for via telefon/brev

**2. Erstatter dyre samarbejdskurser med løbende digital støtte**
- I dag: forældre sendes til kurser (12-13 par pr. kursus, begrænset kapacitet)
- Med Huska: forældrene har et permanent digitalt værktøj der understøtter samarbejde 24/7
- Appen er kursets forlængelse — ikke en erstatning, men et dagligt redskab

**3. Professionelt dashboard til sagsbehandlere (allerede bygget)**
- Sagsoverblik med prioritet og risikovurdering
- Mødereferater med godkendelsesworkflow
- Aktivitetslog og kontakthistorik
- Departement-baseret sagsfordeling

**4. Dokumentation og gennemsigtighed**
- Alt kommunikation logges → kan bruges som dokumentation
- Beslutningslog med tidsstempel og godkendelse
- Afleveringschecklist sikrer børnenes behov

**5. Børnefokus**
- Rutiner og dagbog sikrer kontinuitet mellem hjem
- Børneoverblik med milepæle og sundhedsdata
- Allergenhåndtering og ernæringssporing

**6. Økonomisk transparens**
- Udgiftsdeling med automatisk beregning
- Kvitteringsfotografering og historik
- Reducerer konflikter om penge

## Konkurrencemæssig position

- Ingen eksisterende dansk co-parenting-app med denne funktionalitet
- Familieretshuset bruger cBrain til intern sagsbehandling — men IKKE til forældrekommunikation
- Eksisterende løsninger: SMS, e-mail, MinRetssag (til retssager), e-Boks (dokumenter) — alt fragmenteret
- Huska samler alt ét sted med professionel integration
